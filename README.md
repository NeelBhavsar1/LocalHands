# LocalHands

A comprehensive service marketplace platform connecting local service providers with customers in their community.

## Features
### Core Functionality
- **Service Marketplace**: Create, manage, and discover local services
- **Multi-language Support**: English, Spanish, French, and German
- **Location-based Discovery**: Find services near you with interactive maps
- **Real-time Messaging**: Secure in-app communication between users
- **Review System**: Verified reviews and ratings for service providers
- **Profile Management**: Comprehensive user profiles with service history

### Key Features
- **Interactive Maps**: Visual service discovery with location filtering
- **Secure Messaging**: Real-time communication between users
- **Community Reviews**: Trustworthy rating system
- **Internationalization**: Full multi-language support
- **Responsive Design**: Mobile-first approach
- **User Authentication**: Secure login and account management
- **Theme Support**: Light and dark mode


## Internationalization
The application supports four languages:

- **English** (`en/common.json`)
- **Spanish** (`es/common.json`)
- **French** (`fr/common.json`)
- **German** (`de/common.json`)

## API Integration

The frontend connects to a RESTful API for:

- **Authentication**: User login, registration, profile management
- **Services**: CRUD operations for service listings
- **Messages**: Real-time messaging between users
- **Reviews**: Review submission and management
- **Uploads**: Image and file handling
